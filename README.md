Решение представляет собой полный перебор.

Для каждого сотрудника выбираются все возможные варианты отпусков в году, с учетом ограничений.
Собирается каждое сочетание отпусков для сотрудника. Среди всех сочетаний выбирается наилучшее. Наилучший вариант высчитывается по критерию (равномерное распределние отпусков по году).
Расчет наилучшего варианта намеренно подпорчен, так как всегда самый распределенный ежегодный отпуск состоит из 4 частей по 7 дней (примерно 1 в каждый квартал).
